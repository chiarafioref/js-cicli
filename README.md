# ESERCIZIO

Dato un array di prodotti e uno di quantità, stampa se il prodotto è esaurito o se le scorte sono basse.

## RAGIONAMENTO

- Imposto un array di prodotti
- Imposto un array con le quantità specifiche per ogni prodotto
  - SE la quantità del prodotto è 0
    - stampo "Esaurito"
  - SE la quantità del prodotto è minore di 3
    - stampo "Scorte basse"
